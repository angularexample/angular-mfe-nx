App Shell

This demonstrates Angular MFE architecture using NX.

This is the app-shell. It uses routing to load 2 remote MFE apps.

All apps share a common UI Library that shows how to share UI components.

To see how the MFE's are loaded, open the Browser Dev Tools - Network - All tab. When you navigate to a remote MFE, you will see a script is loaded containing the run time MFE code from Webpack.
